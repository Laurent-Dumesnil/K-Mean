Configurations fonctionnelles : 

Regénération de la bd : main.py -b

Entrainement : main.py -e -t(int) --encodage(string) --chemin(string)

Prédiction : main.py -p -t(int)

Cluster : main.py -c -t(int) -k(int) -n(int)
Options possibles avec le cluster : --normaliser(store_true) --conserver(int)
Génération du graphe : --graphe(store_true)
